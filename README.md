# 我的餐廳清單
![Index page about Restaurant List](./public/image/restaurant_list.png)

## 介紹
紀錄餐廳的清單，能夠新增、修改及刪除。

## 功能
- 查看所有餐廳
- 瀏覽餐廳的詳細資訊
- 新增餐廳
- 編輯餐廳
- 刪除餐廳

## 使用說明

1. 請先確認有安裝 node.js 與 npm
2. 將專案 clone 到本地
3. 在本地開啟之後，透過終端機進入資料夾，輸入：

   ```bash
   npm install
   ```

4. 安裝完畢後，繼續輸入：

   ```bash
   npm run start
   ```

5. 若看見此行訊息則代表順利運行，打開瀏覽器進入到以下網址

   ```bash
   Listening on http://localhost:3000
   ```

6. 若欲暫停使用

   ```bash
   ctrl + c
   ```
